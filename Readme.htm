Mini Sample Return Capsule (MSRC)

Two part size zero (0.625m) autonomous return capsule to stuff your experiments in for Kerbal Space Program.

Adopted for curation by @zer0Kerbal, originally by @CobaltWolf and @AlbertKermin

brought to you by KGEx

Description

Preamble by @CobaltWolf

Hello all! I have another new mod for you. A couple weeks ago, the incredibly handsome @AlbertKermin asked me if I was interested in making a mini-mod. Inspired by the OSIRIS-Rex mission, he wanted a small sample return capsule that could be loaded with experiments and safely return.

Features

This mod adds two parts

  • Part: command/parachute
  • Part: heatshield/cargobay
  • RemoteTech configs are included.


Cabin notes

  • The mod contains two parts, a command/parachute and Heatshield/cargo-bay.
  • Future Plans: lol. Except compatibility patches, keep it working with future KSP updates.
  • Community Tech Tree integration and Tech Tree Balancing (need feedback and assistance)
  • I have not personally tested the patches
  • Have a request? Glad to have them, kindly submit through GitHub.
  • Would love to see one or more textures that could be set as variants. Would need help with the textures. :D
